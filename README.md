## Event loop nodejs

La tecnología nodejs se basa en el uso de eventos, por eso es mas rápido que otras tecnologías.
En una aplicación de eventos, existe un main loop, o ciclo central el cual espera por eventos, y manda a llamar
una función callback cuando uno de éstos eventos es detectado.
